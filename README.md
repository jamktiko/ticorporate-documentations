# Ticorporate Documentation
This repository contains templates, essential documents, and general information for Jamk Business Information Technology courses operating under the Ticorporate brand, supporting students with consistent materials, shared guidelines, and practical resources.
## Sitemap

This directory contains default templates for everything. Some courses have a specific templates, which is mentioned in Moodle or Teams. This "special templates" can be found course specific directories.
